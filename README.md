# Demo Dart application using NodeJS as runtime

See implementation in `node/demo.dart`.

## Installation:

```bash
$ git clone
$ cd dartconf2018/
$ pub get
$ # Compile with DDC
$ pub run build_runner build --output=build/
```

## Running:

```
$ node build/node/demo.dart.js
```
